## Project Structure

- `token-ui/` - React Next.js frontend application
- `token-contract/` - Solidity smart contracts and tests

## Smart Contract Features

- ERC20 standard implementation
- Ownership with transfer functionality
- Minting restricted to contract owner
- Proper event emission

## Frontend Features

- Wallet connection via MetaMask
- Token balance display
- Transfer functionality
- Minting interface for owners
- Add token to wallet (EIP-747)

## Technologies Used

- **Frontend**: React, Next.js, Web3.js
- **Smart Contracts**: Solidity, Hardhat
- **Testing**: Hardhat, Viem
- **Network**: Ethereum Sepolia Testnet

## Getting Started

### Prerequisites
- Node.js
- MetaMask
- Hardhat

### Installation
1. Clone the repository
2. Install dependencies in both folders
3. Deploy contracts
4. Start the frontend
